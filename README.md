# news-bcs-nuxt

> Show BCS express news.

[news-bcs.levitatum.ru](http://news-bcs.levitatum.ru/)

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```
